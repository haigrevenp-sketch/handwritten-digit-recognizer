Overview

Project Overview

This application demonstrates how Convolutional Neural Networks can achieve near-human accuracy on handwritten digit classification using the famous MNIST dataset. Built as a portfolio-grade project, it combines a trained deep-learning model with a modern Flask web interface featuring real-time prediction, Explainable AI (Grad-CAM), PDF reports, SQLite persistence, and a responsive dark/light-mode UI.

  • CNN trained to ~99.3% test accuracy
  • Real-time canvas prediction
  • Grad-CAM heatmap visualisation
  • Downloadable PDF prediction reports
  • Prediction history saved in SQLite
  • Dark / Light mode responsive UI
